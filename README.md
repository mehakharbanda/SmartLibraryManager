Smart Library Manager
Live Demo: https://beeproject-6.onrender.com/

Overview
SLM is a full-stack web application built to streamline and digitalize essential library operations with a user-friendly interface and a secure backend. The project combines a React-based frontend for smooth interaction with an Express.js + Node.js backend that handles authentication, business logic, and database operations. Data is stored securely in MongoDBL, ensuring efficient management and retrieval.

The application provides:

User Accounts & Authentication: Secure login and signup using JWT tokens.
Data Management: CRUD (Create, Read, Update, Delete) functionality for managing resources like books, products, employees, or reservations.
Borrowal Records: Track issue and return history of books with due dates.
Responsive Experience: A modern and mobile-friendly UI that adapts to devices of all sizes.
Seamless Deployment: Hosted on Render, making it accessible anytime, anywhere.
Tech Stack
Frontend: HTML, CSS, JavaScript, React.js
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT
Hosting/Deployment: Render
