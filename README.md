 Live Demo:
https://beeproject-6.onrender.com/

## Overview
Smart Library Manager (SLM) is a full-stack web application designed to digitalize library operations with a clean UI and a secure backend.
It's uses:
A React.js frontend for smooth and responsive user interactions
An Express.js + Node.js backend for handling authentication, routing, and business logic
MongoDB for secure and efficient data storage

## Features
1.User Accounts & Authentication-Secure Login & Signup with JWT Authentication

2.Data Management:Full CRUD (Create, Read, Update, Delete) operations&Manage books, products, employees, members, reservations, etc.

3.Borrowal Records
Track issue & return history
Maintain due dates and borrower logs

4.Responsive Experience
Modern UI
Works seamlessly on mobile, tablet, and desktops

5.Seamless Deployment
Hosted on Render for easy accessibility
